# 💣 Hangman Game — x86 Assembly (COAL)

A fully functional **Hangman game** written in **x86 Assembly language (MASM)**, developed as a 4th semester Computer Organization and Assembly Language (COAL) project at Air University.

This project demonstrates low-level programming skills including memory management, registers, and system interrupts — all without using any high-level language.

---

## 🎮 Game Features

- 🔤 Random word selection for each game session
- 💡 Letter guessing with visual feedback
- ❤️ Lives system — limited incorrect guesses allowed
- 🖥️ Console-based UI rendered using Assembly routines
- 🔁 Play again option after each round

---

## 🧠 COAL Concepts Used

- **Registers** — AX, BX, CX, DX for data manipulation
- **Interrupts** — DOS/BIOS interrupts for I/O operations
- **Procedures** — Modular code using PROC and ENDP
- **String Handling** — Manual character comparison and display
- **Conditional Jumps** — JE, JNE, JMP for game logic
- **Memory Addressing** — Direct and indirect memory access

---

## 🛠️ Tech Stack

- **Language:** x86 Assembly (MASM syntax)
- **Assembler:** MASM / EMU8086
- **Platform:** DOS / DOSBox

## ⚙️ How to Run

1. Clone the repository:
```bash
   git clone https://github.com/muhammadbilalkhalidmbk/hangman-assembly.git
```
2. Open `hangman.asm` in **EMU8086** or assemble using **MASM**
3. Run inside **DOSBox** or EMU8086 emulator

> 💡 Recommended: Use [EMU8086](https://emu8086-microprocessor-emulator.en.softonic.com/) for easiest setup on Windows.
